🧙 Průvodce instalací

1

Připrav Home Assistant

Tento průvodce popisuje Scénář A — Provisioning: standardní Raspberry Pi OS Lite z RPi Imager + automatická konfigurace při prvním startu.

Než začneš, vytvoř v HA dedikovaného uživatele pro kiosk:

Vytvoř uživatele HA → Nastavení → Lidé → Přidat osobu
Uživatelské jméno Malá písmena, číslice, podtržítko — např. kiosk1
Vygeneruj token Přihlas se jako tento uživatel → Profil → Bezpečnost → Dlouhodobé přístupové tokeny → Vytvořit token → zkopíruj
💡 Proč samostatný uživatel? Token v kiosk.conf má plný přístup k HA. Dedikovaný účet s omezenými právy je bezpečnější než token hlavního admina.
2

Vygeneruj kiosk.conf

Přejdi na Přidat kiosk a vyplň konfiguraci pro nový kiosk.

➕ Přidat kiosk

Po uložení se automaticky stáhne soubor kiosk.conf.

🔑 HA uživatel a token: Do formuláře zadej uživatelské jméno a token vytvořený v Kroku 1. Token se generuje v profilu kiosk uživatele v HA (Profil → Bezpečnost → Dlouhodobé přístupové tokeny).
3

Flashuj SD kartu v RPi Imager

Stáhni a otevři Raspberry Pi Imager:

Choose Device → Raspberry Pi 5
Choose OS → Raspberry Pi OS (other) → Raspberry Pi OS Lite (64-bit)
Choose Storage → Vyber SD kartu (min. 16 GB)
Next → Edit Settings → Nastav:
Hostname kiosk (nebo specifický název)
Username pi ← musí být přesně pi!
Password Zvol si heslo (ulož si ho!)
SSH ✓ Enable SSH (Password authentication)
WiFi LAN kiosk: ❌ NEVYPLŇUJ — připojení přes kabel
WiFi kiosk: ✓ VYPLŇ SSID + heslo — bez toho se RPi nepřipojí k síti

Klikni Save → Yes → Yes a počkej na dokončení flashování.

4

Zkopíruj kiosk.conf na SD kartu

Po flashování bude boot oddíl SD karty viditelný ve Průzkumníkovi (Windows) nebo jako /media/boot (Linux).

  1. Otevři boot oddíl SD karty
  2. Zkopíruj stažený kiosk.conf přímo do kořene boot oddílu
  3. Vedle souborů config.txt, cmdline.txt atd.
  4. Bezpečně odejmi SD kartu
⚠️ Bezpečnost: kiosk.conf obsahuje HA token. Firstboot.sh ho automaticky smaže po konfiguraci. Nikomu nesdílej kiosk.conf soubor!
5

Připrav Raspberry Pi

Vlož SD kartu do RPi 5, připoj LAN kabel (nebo zajisti WiFi dle Kroku 3) a zapni napájení.

🟢 Boot RPi OS
↓
🔍 Firstboot detekuje kiosk.conf
↓
⚙️ Nastaví hostname, síť, rozlišení
↓
📡 Phone-home → Kiosk Builder addon
↓
🔑 Uloží SSH klíč
↓
🗑 Smaže kiosk.conf (bezpečnost!)
↓
🔄 Reboot
↓
📺 Chromium → HA Dashboard

Celý proces trvá 5–10 minut.

Sleduj log: ssh pi@KIOSK-IP "tail -f /var/log/kiosk-firstboot.log"

6

✅ Hotovo!

Po druhém restartu by se měl kiosk zobrazit jako Online v dashboardu.

📋 Zobrazit dashboard kiosků
🛠 Vzdálená správa:
• VNC: RPi IP:5900 (přes RealVNC Viewer)
• SSH: ssh pi@KIOSK-IP
• Glances: http://KIOSK-IP:61208
